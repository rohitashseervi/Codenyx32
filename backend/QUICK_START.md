# GapZero Backend - Quick Start Guide

## Setup (5 minutes)

```bash
cd /sessions/nifty-magical-gauss/mnt/Hackathon/gapzero/backend

# 1. Install dependencies
npm install

# 2. Setup environment
cp .env.example .env
# Edit .env with your credentials

# 3. Start MongoDB
mongod

# 4. Run server
npm run dev
```

Server runs on: `http://localhost:5000/api`

## Core Flows

### 1. Register & Login
```bash
# Register
curl -X POST http://localhost:5000/api/auth/register \
  -H "Content-Type: application/json" \
  -d '{
    "uid": "firebase-uid",
    "email": "student@example.com",
    "displayName": "John Doe",
    "role": "student"
  }'

# Login
curl -X POST http://localhost:5000/api/auth/login \
  -H "Content-Type: application/json" \
  -d '{"token": "firebase-id-token"}'
```

### 2. Request Mentor (Smart Matching)
```bash
curl -X POST http://localhost:5000/api/student/request-mentor \
  -H "Authorization: Bearer <token>"
# Returns: Matched mentor with match score (0-100)
```

### 3. Take Test (Auto-generated by AI)
```bash
# Get pending tests
curl -X GET http://localhost:5000/api/student/tests \
  -H "Authorization: Bearer <token>"

# Submit test
curl -X POST http://localhost:5000/api/student/tests/<testId>/submit \
  -H "Authorization: Bearer <token>" \
  -H "Content-Type: application/json" \
  -d '{"answers": ["A", "B", "C", ...]}'
# Returns: Score, badge rewards, detailed results
```

### 4. Dashboard Analytics
```bash
curl -X GET "http://localhost:5000/api/dashboard/overview?ngoId=<ngoId>" \
  -H "Authorization: Bearer <token>"
# Returns: KPIs, at-risk count, health score
```

## Troubleshooting

**MongoDB connection fails**
- Ensure mongod is running: `mongod`
- Check MONGODB_URI in .env

**Firebase auth fails**
- Verify credentials in .env
- Ensure Firebase Admin SDK initialized

**OpenAI test generation fails**
- API falls back to pre-built questions
- Check OPENAI_API_KEY in .env

**Gmail notifications not sending**
- Verify GMAIL_USER and GMAIL_APP_PASSWORD
- Gmail account must have 2FA enabled
- Use app-specific password (not account password)

**Google Meet links not creating**
- Ensure service account has Calendar API enabled
- Check GOOGLE_SERVICE_ACCOUNT_KEY_PATH points to valid JSON
